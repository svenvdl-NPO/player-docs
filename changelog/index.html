Changelog

This page only contains the changelog for the Web library of the NPO Player.

1.1.9 (2023-02-07)

Added

  • Chromecast support for our own custom receiver enabling more control over the user experience and future tracking capabilities.
  • ‘Up next’ feature that automatically plays a next stream if a PRID is provided by the API.
  • A conditional .ui-livestream class is now set on the player container if the stream is recognised as a livestream by the API for more flexibility in styling.

Changed

  • Refactored SCSS for improved maintainability.

  • Fixed an issue where firefox would not display video when certain buttons were removed from the UI during playback.

  • Some redundant static files have been removed from the library.
