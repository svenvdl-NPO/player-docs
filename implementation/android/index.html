Adding the library to your own app

In you projects build.gradle or settings.gradle you will need to add this maven repository:

// NPO Player library
maven {
    url 'https://pkgs.dev.azure.com/NPOplayer/NPO-player-public/_packaging/NPOPlayerLibrary-Android/maven/v1'
    name 'NPOPlayerLibrary-Android'
}
// Bitmovin Player
maven {
    url 'https://artifacts.bitmovin.com/artifactory/public-releases'
}

Then in you app’s build.gradle import it through:

    // -- NPOPlayer library --//
implementation "nl.npo.player.library:npoPlayerAndroid:<LATEST_VERSION>"

Optionally your different modules might need access to a subset of this data, like the domain or NPOTag specific data. Then import those libraries separately:

implementation "nl.npo.player.library:domain:<LATEST_VERSION>"
implementation "nl.npo.player.library:npoTag:<LATEST_VERSION>"

Check out the "<LATEST_VERSION>" number from Azure.

When the library package is added to your project you will need to initialise the library before you use it by providing it with an (Application) Context an Environment (TEST, ACC, PROD) and either an NpoTag or AnalyticsConfiguration:

NPOPlayerLibrary.initialize(context = Context, environment = Environment, npoTag = NpoTag)
NPOPlayerLibrary.initialize(
    context = Context,
    environment = Environment,
    analyticsConfiguration = AnalyticsConfiguration
)

Optionally with either of these initializers you can also add a list of OkHttp3 Interceptor’s for debugging purposes through the optionalInterceptors parameter (for example so you can log the network call or add a Chucker interceptor).

Then for playing streams you will need to instantiate a NPOPlayerBitmovin which needs a Context, PlayerPageTracker and optionally a NPOPlayerConfig. To get a PlayerPageTracker you need to make a call to the helper object PlayerTagProvider getPageTracker function. You can either supply is an NpoTag PageTracker (if your app uses it’s own NpoTag implementation) or a PageConfiguration.

AndroidTV

The AndroidTV module is still very much Work In Progress but can separately be loaded in in a Leanback application using:

implementation "nl.npo.player.library:domain:<LATEST_VERSION>"
implementation "nl.npo.player.library:presentationTV:<LATEST_VERSION>"
