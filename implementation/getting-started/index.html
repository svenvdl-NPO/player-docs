Welcome > Implementation > Getting started

Bitmovin

Before you can start using the NPO Player you need access to the Bitmovin Dashboard. In order to get access please contact Leon Kohlen leon.kohlen@npo.nl.

  1. Once you access to the dashboard you can find your player and analytics key. These keys are mandatory and the player will not work without them.

  2. Make sure your domain/app is whitelisted.

NPOTag

Next you will need to get NPOTag credentials if you haven’t implemented V2 of the NPOTag yet. Get more information on the NPOTag (including how to request credentials) here: https://npotag.npo-data.nl/docs/index.html The minimum information you need is brand and brandId. You can either supply the player/library with an NpoTag implementation which you have already instantiated (for your own website/app usage tracking) or with a configuration object so a NpoTag instance can be created in the library.

JWT

Finally to start a stream we need to create a JSON web token (JWT). Continue creating JWT.
